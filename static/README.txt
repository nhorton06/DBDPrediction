LOGO INSTRUCTIONS
=================

To add the Dead by Daylight logo:

1. Find or download the DBD logo image file
2. Save it as "dbd_logo.png" in this folder (static/)
3. The logo will automatically appear on the website

If no logo is found, the website will display "DEAD BY DAYLIGHT" text instead.

Supported formats: PNG, JPG, SVG
Recommended size: 300px width or larger (will be resized automatically)
